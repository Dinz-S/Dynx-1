# Dnyx To-Do List

A simple To-Do List application where users can:
- Add tasks  
- Edit tasks  
- Delete tasks  
- Mark tasks as completed  
- Filter tasks (All, Active, Completed)  
- Store tasks in LocalStorage (data remains after refresh)
